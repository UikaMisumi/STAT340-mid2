## 一、课堂练习回顾：置信区间相关考题

### 练习1：FA25 期中考 第4题

**题目**：从几何随机变量中抽取大小为16的随机样本，构造参数 $p$ 的95%置信区间为 $(0.456, 0.563)$，判断以下哪些正确。

**逐项分析**：

1. **"有95%的概率 $0.456 \le p \le 0.563$"** — ❌ 错误。$p$ 是固定的总体参数（deterministic），不是随机变量。置信区间是随机的，参数不随机。

2. **"若样本量翻倍，区间宽度约为原来一半"** — ❌ 错误。方差与样本量成反比，标准差与 $\sqrt{n}$ 成反比，因此样本量翻倍时区间宽度缩窄为原来的 $1/\sqrt{2}$ 倍（约0.707倍），而非一半。

3. **"若对 $H_0: p = 0.5$ 做双侧检验，在 $\alpha = 0.05$ 下不会拒绝 $H_0$"** — ✅ **正确**。因为 $0.5$ 落在置信区间 $(0.456, 0.563)$ 内，根据置信区间与假设检验的对偶性（duality），不会拒绝。

4. **"若重新取样，有95%概率新区间与此区间重叠"** — ❌ 错误。置信区间的定义是"有95%概率覆盖真实参数"，不涉及两个独立区间是否重叠。

5. **"样本均值 $\bar{x} \in (0.456, 0.563)$"** — ❌ 错误。几何分布中 $p$ 的点估计不是 $\bar{x}$（而是与 $\bar{x}$ 的倒数相关），置信区间的中心也不一定等于样本均值。

### 练习2：S25 期中考 MC9

**题目**：$n=20$，Poisson总体，基于10,000次模拟构造 $\lambda$ 的95% Monte Carlo置信区间为 $(0.45, 0.72)$，选最正确选项。

**正确答案**：**(a)** — 随机挑选一个MC模拟点估计，有95%的概率落在 $(0.45, 0.72)$ 内。

**解释**：
- 回顾模拟置信区间构造步骤：先估计 $\hat{\lambda}$，再从 $\text{Poisson}(\hat{\lambda})$ 生成10,000批数据，每批计算点估计，最后取2.5%和97.5%分位数。所以区间恰好覆盖这些模拟值的中间95%。
- **(b)(c)** 错误：$\lambda$ 是固定参数，不能说"有xx%概率 $\lambda$ 在某区间"。
- **(d)** 错误：无法保证重新抽样的样本均值一定落在此区间。
- **(e)** 错误：增加MC次数（如100,000）不会使区间变窄。更多模拟只是让经验分布更接近真实抽样分布，而非让分布集中到一个点。

## 二、预测问题概述

### 2.1 预测是机器学习的核心任务

- **图像分类**：给定图片，预测是猫还是狗
- **推荐系统**：预测用户对新内容的参与度（engagement）

### 2.2 抽象框架

给定训练数据 $(X_1, Y_1), \ldots, (X_n, Y_n)$：
- $X_i$：**预测变量**（predictors / independent variables）— 用于预测的特征
- $Y_i$：**响应变量**（response / dependent variable / outcome）— 需要预测的目标

> **注意**：称 $X$ 为"自变量"并不意味着各 $X_i$ 之间概率独立，而是因为我们不关心 $X$ 的分布，只关心 $Y$ 如何依赖 $X$。类似地，$Y$ 被称为"因变量"因为我们要研究 $Y$ 对 $X$ 的依赖关系。

## 三、线性回归导论

### 3.1 什么是线性回归

线性回归是一种强大的统计工具，用于建模和分析响应变量 $Y$ 与一个或多个解释变量 $X$ 之间的**线性关系**。

应用场景包括：医疗（预测患者结果）、营销（基于广告支出估计销售额）、环境科学（分析污染物对健康的影响）等。

### 3.2 简单 vs 多元线性回归

| 类型 | 说明 |
|------|------|
| **简单线性回归** (Simple) | $X$ 是一维的（一个自变量） |
| **多元线性回归** (Multiple) | $X$ 是多维的（多个自变量） |

本节从简单线性回归开始讲起。

---

## 四、简单线性回归模型

### 4.1 模型定义

$$Y_i = \beta_0 + \beta_1 X_i + \epsilon_i, \quad i = 1, \ldots, n$$

- $\beta_0$（截距 intercept）：当 $X = 0$ 时 $Y$ 的均值
- $\beta_1$（斜率 slope）：$X$ 每增加一个单位，$Y$ 均值的变化量
- $\epsilon_i$（误差项 error term）：第 $i$ 个观测的噪声，代表模型无法解释的因素

**直觉理解**：给定预测变量值 $X_i = x_i$，$Y_i$ 的"真实"信号部分是 $\beta_0 + \beta_1 x_i$，但实际观测值还受到噪声 $\epsilon_i$ 的影响。$\epsilon_i$ 包含了数据集中未记录的所有其他因素。

### 4.2 示例：教育年限与收入

- $(X_i, Y_i)$：教育年限 vs 收入（万美元）
- 假设拟合得到 $Y_i = 20 + 4X_i$
- **解释**：教育年限每增加1年，模型预测收入**关联性地**增加4万美元；0年教育对应预测收入2万美元

### 4.3 模型假设

1. **线性性 (Linearity)**：$\mathbb{E}[Y | X = x] = \beta_0 + \beta_1 x$
2. **独立性 (Independence)**：$\{\epsilon_i\}_{i=1}^n$ 是独立的随机变量序列
3. **同方差性 (Homoscedasticity)**：$\text{Var}(Y_i | X_i) = \sigma^2$ 对所有 $i$ 相同
4. **（可选）正态误差**：$\epsilon_i \sim N(0, \sigma^2)$，即 $Y_i | X_i \sim N(\beta_0 + \beta_1 X_i, \sigma^2)$

### 4.4 模型参数

共三个未知参数：
- $\beta_0$：截距（回归系数之一）
- $\beta_1$：斜率（回归系数之一）
- $\sigma^2$：噪声方差（也写作 $\sigma^2_\epsilon$ 或 $\sigma^2_{Y|X}$）

> **重要提醒**：线性模型是我们的**假设**，不一定是真实世界的真相。只有在线性假设近似成立时，模型才能给出好的预测。

## 五、注意事项

### 5.1 $X$ 的适用范围

- 模型只在训练数据覆盖的 $X$ 范围内有意义
- 例如用身高预测收入，$\hat{\beta}_0 = 10$ 不能解读为"身高为0的人年收入1万美元"——现实中不存在这样的 $X$ 值
- **外推（extrapolation）要谨慎**

### 5.2 因果关系 vs 关联关系

- 线性回归只能揭示**关联关系（association）**，不能证明**因果关系（causation）**
- 例如 $\hat{\beta}_1 = 10.2$（PM2.5 → 癌症病例）：只能说 PM2.5 每增加一个单位**关联**增加10.2例癌症，**不能**说"导致"
- 这就是经典的 "Correlation is not causation"

## 六、模型拟合：最小二乘估计 (OLS)

### 6.1 核心思想

给定散点图上的数据点，我们要找一条直线使得所有点到这条线的**残差平方和最小**。

**损失函数**（残差平方和 / RSS / SSE）：

$$Q(\beta_0, \beta_1) = \sum_{i=1}^{n} (Y_i - \beta_0 - \beta_1 X_i)^2$$

**残差**定义为：$e_i = y_i - \hat{y}_i = y_i - (\beta_0 + \beta_1 x_i)$，即观测值与预测值的垂直距离。

### 6.2 为什么用平方而非绝对值

另一种可能的损失函数是绝对偏差之和 $\sum |y_i - \hat{y}_i|$，但它不光滑（not smooth），优化更困难。平方损失是强凸且光滑的，有解析解。

### 6.3 求解：偏导数为零

对 $Q(\beta_0, \beta_1)$ 分别对 $\beta_0$ 和 $\beta_1$ 求偏导并令其为零：

$$\frac{\partial Q}{\partial \beta_0} = 0, \quad \frac{\partial Q}{\partial \beta_1} = 0$$

**闭式解**：

$$\hat{\beta}_1 = \frac{\sum_{i=1}^{n}(x_i - \bar{x})(y_i - \bar{y})}{\sum_{i=1}^{n}(x_i - \bar{x})^2}$$

$$\hat{\beta}_0 = \bar{y} - \hat{\beta}_1 \bar{x}$$

其中 $\bar{x} = \frac{1}{n}\sum x_i$，$\bar{y} = \frac{1}{n}\sum y_i$。

### 6.4 估计量的解读

**$\hat{\beta}_1$ 的含义**：

$$\hat{\beta}_1 = \frac{s_y}{s_x} \cdot \hat{\rho}_{x,y}$$

即模型斜率 = 响应变量标准差与预测变量标准差之比 × 样本相关系数。

- 当数据完全线性（无抖动），$\hat{\rho}_{x,y} = 1$，此时 $\hat{\beta}_1 = s_y / s_x$（纯粹的单位转换）
- 其中样本相关系数 $\hat{\rho}_{x,y} = \frac{\frac{1}{n}\sum(x_i - \bar{x})(y_i - \bar{y})}{\sqrt{s_x^2 \cdot s_y^2}}$

**$\hat{\beta}_0$ 的含义**：

- $\hat{\beta}_0 = \bar{y} - \hat{\beta}_1 \bar{x}$ 意味着**回归线总是经过数据中心 $(\bar{x}, \bar{y})$**
- 截距的作用是在已确定斜率后，平移直线使其最好地拟合数据

### 6.5 $\hat{\beta}_1$ 的方差

$$\text{Var}(\hat{\beta}_1) = \frac{\sigma^2}{\sum_{i}(x_i - \bar{x})^2}$$

**推导要点**：
1. 将 $y_i = \beta_0 + \beta_1 x_i + \epsilon_i$ 代入 $\hat{\beta}_1$ 表达式
2. 利用 $\sum(x_i - \bar{x}) = 0$ 消去常数项
3. 利用 $\epsilon_i$ 独立且 $\text{Var}(\epsilon_i) = \sigma^2$
4. 利用 $\text{Var}(cX) = c^2 \text{Var}(X)$ 化简

**直觉**：$X$ 的分散程度越大（$\sum(x_i - \bar{x})^2$ 越大），斜率估计越精确（方差越小）。
