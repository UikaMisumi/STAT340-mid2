## 一、点估计回顾（Point Estimation Review）

### 1.1 基本概念

**点估计**是用样本数据计算一个单一值来估计/猜测未知总体参数 $\theta(P)$ 的统计方法。

**核心流程**：
$$\text{未知总体 } P \xrightarrow{\text{抽样}} X_1, X_2, \ldots, X_n \xrightarrow{\text{用函数汇总}} \hat{\theta}(X_1, \ldots, X_n)$$

- **目标参数** $\theta(P)$：总体的某个未知性质，例如总体均值 $\mu$
- **估计量（Estimator）**：样本的函数 $\hat{\theta}(X_1, \ldots, X_n)$，是**随机变量**，不能包含任何未知参数
- **估计值（Estimate）**：将实际观测值代入后得到的**具体数值** $\hat{\theta}(x_1, \ldots, x_n)$
- **抽样分布（Sampling Distribution）**：估计量作为随机变量所服从的分布

> **老师强调**：估计量（estimator）是抽象的函数形式，处理的是随机变量；估计值（estimate）是代入实际数据后得到的具体数字。两者概念不同但经常混用。

### 1.2 统计量（Statistic）

统计量是样本的任意函数，用于汇总数据集的某些特征（如均值、中位数、分位数、顺序统计量等）。估计量本身也是一种统计量。

---

## 二、好估计量的三大标准

给定 $X_1, \ldots, X_n \overset{\text{iid}}{\sim} P$，估计量 $\hat{\theta}(X_1, \ldots, X_n)$ 的好坏从三个不同角度衡量：

### 2.1 无偏性（Unbiasedness）

$$\mathbb{E}[\hat{\theta}(X_1, \ldots, X_n)] = \theta$$

- 这是一个**有限样本**性质：只要当前样本量 $n$ 下期望恰好等于真实参数，就是无偏的
- 不需要考虑 $n$ 增大时的行为
- **直觉**：估计量有时高估、有时低估，但**平均而言不偏离**真实值

> **老师强调**：无偏并不意味着估计量一定好——估计量可能无偏但方差极大，导致单次估计远离真实值。

### 2.2 一致性（Consistency）

$$\text{当 } n \to \infty,\quad \hat{\theta}(X_1, \ldots, X_n) \xrightarrow{} \theta$$

- 这是一个**渐近**性质：要求样本量趋于无穷时估计量收敛到真实参数
- 比无偏性更抽象，因为它需要考虑极限行为
- 样本均值、样本方差、样本标准差等常见估计量都满足一致性（由大数定律保证）

### 2.3 均方误差（Mean Squared Error, MSE）

$$\text{MSE}(\hat{\theta}) := \mathbb{E}[(\hat{\theta}(X_1, \ldots, X_n) - \theta)^2]$$

- 衡量估计量与真实参数之间**平均平方距离**
- MSE 小意味着估计量在有限样本下平均而言离真实值很近
- 可以看作一致性的**有限样本代理**

> **三个标准互不等价**：任何两个标准度量的角度不同。一个估计量可能满足其中一个或多个，但满足一个不自动满足另一个。

## 三、MSE 的偏差-方差分解

### 3.1 分解公式

$$\boxed{\text{MSE}(\hat{\theta}) = \text{Bias}(\hat{\theta})^2 + \text{Var}(\hat{\theta})}$$

其中 $\text{Bias}(\hat{\theta}) = \mathbb{E}[\hat{\theta}] - \theta$。

### 3.2 推导过程（老师板书讲解）

$$\text{MSE}(\hat{\theta}) = \mathbb{E}[(\hat{\theta} - \theta)^2]$$

将 $\hat{\theta} - \theta$ 分解为两部分：

$$\hat{\theta} - \theta = (\hat{\theta} - \mathbb{E}[\hat{\theta}]) + (\mathbb{E}[\hat{\theta}] - \theta)$$

- 第一项 $(\hat{\theta} - \mathbb{E}[\hat{\theta}])$ 描述估计量的**离散程度**（方差项）
- 第二项 $(\mathbb{E}[\hat{\theta}] - \theta)$ 描述估计量的**系统偏差**（偏差项，是常数）

展开平方后取期望：
$$\mathbb{E}[(\hat{\theta} - \mathbb{E}\hat{\theta})^2] + (\mathbb{E}\hat{\theta} - \theta)^2 + 2(\mathbb{E}\hat{\theta} - \theta)\cdot \underbrace{\mathbb{E}[\hat{\theta} - \mathbb{E}\hat{\theta}]}_{= 0}$$

交叉项为零，因此得到 $\text{Var}(\hat{\theta}) + \text{Bias}(\hat{\theta})^2$。

### 3.3 重要推论

- 若 $\hat{\theta}$ 无偏（Bias = 0），则 $\text{MSE}(\hat{\theta}) = \text{Var}(\hat{\theta})$
- MSE 小同时要求偏差和方差都不大——二者需要**权衡（tradeoff）**

## 四、练习与实例

### 4.1 练习：估计 $\mu^2$ 的偏差

**设置**：$X_1, \ldots, X_n \sim N(\mu, \sigma^2)$，用 $T = X_1^2$ 估计 $\mu^2$。

**偏差计算**：
$$\text{Bias} = \mathbb{E}[X_1^2] - \mu^2 = (\text{Var}(X_1) + (\mathbb{E}X_1)^2) - \mu^2 = \sigma^2 + \mu^2 - \mu^2 = \sigma^2$$

> **关键技巧**：$\mathbb{E}[X^2] = \text{Var}(X) + (\mathbb{E}X)^2$，这是一个非常常用的恒等式。

### 4.2 实例：$S^2$ vs $\hat{\sigma}^2$ 的偏差-方差权衡

设 $X_1, \ldots, X_n \sim N(\mu, \sigma^2)$，比较两个方差估计量：

| | $S^2 = \frac{1}{n-1}\sum(X_i - \bar{X})^2$ | $\hat{\sigma}^2 = \frac{1}{n}\sum(X_i - \bar{X})^2$ |
|---|---|---|
| **无偏性** | $\mathbb{E}[S^2] = \sigma^2$（无偏） | $\mathbb{E}[\hat{\sigma}^2] = \frac{n-1}{n}\sigma^2$（有偏） |
| **偏差** | $0$ | $-\sigma^2/n$ |
| **方差** | $\frac{2\sigma^4}{n-1}$ | $\frac{2(n-1)}{n^2}\sigma^4$ |
| **MSE** | $\frac{2\sigma^4}{n-1}$ | $\frac{2n-1}{n^2}\sigma^4$ |

**结论**：$\text{MSE}(\hat{\sigma}^2) < \text{MSE}(S^2)$

> **核心启示**：$S^2$ 无偏但方差更大；$\hat{\sigma}^2$ 有偏但方差更小。最终 $\hat{\sigma}^2$ 的 MSE 更小。**无偏不等于最小 MSE**。这就是偏差-方差权衡的具体体现。

### 4.3 练习：样本均值的 MSE

$X_1, \ldots, X_n \sim N(\mu, \sigma^2)$，估计量 $\hat{\mu} = \bar{X} = \frac{1}{n}\sum X_i$。

- $\hat{\mu}$ 是无偏的：$\mathbb{E}[\hat{\mu}] = \mu$
- 对于无偏估计量：$\text{MSE}(\hat{\mu}) = \text{Var}(\hat{\mu}) = \frac{\sigma^2}{n}$

## 五、偏差-方差权衡在模型中的推广

在机器学习的预测问题中同样存在偏差-方差权衡：

| 模型复杂度 | 偏差 | 方差 | 状态 |
|---|---|---|---|
| 过低（过于简化） | 高 | 低 | 欠拟合（Underfitting） |
| 过高（过于复杂） | 低 | 高 | 过拟合（Overfitting） |
| 适中 | 适中 | 适中 | **最优模型** |

> **老师解释**：过于简单的模型（如用线性拟合实际上的二次关系）会引入高偏差但预测稳定（低方差）；过于复杂的模型（包含大量参数）偏差小但在有限样本上方差大，样本外预测会"爆炸"。最优模型在二者之间取得平衡。

## 六、矩估计法（Method of Moments, MoM）

### 6.1 一般步骤

给定 $X_1, \ldots, X_n$，要估计参数 $\theta(P) \in \mathbb{R}$：

**第一步：计算矩**

选择 $k \in \mathbb{N}$（通常 $k = 1$ 或 $k = 2$），使得 $\mathbb{E}_{X \sim P}[X^k] = g(\theta)$ 容易计算。

**第二步：匹配矩，构造估计量**

由大数定律，$\frac{1}{n}\sum_i X_i^k \approx \mathbb{E}[X^k] = g(\theta)$。

令 $g(\hat{\theta}) = \frac{1}{n}\sum_i X_i^k$，解出 $\hat{\theta}$ 即为矩估计量。

> **核心逻辑**：$g(\hat{\theta}) = \frac{1}{n}\sum X_i^k \approx g(\theta)$，因此 $\hat{\theta} \approx \theta$。

### 6.2 实例：指数分布的速率参数

$X_1, \ldots, X_n \sim \text{Exp}(\lambda)$，$\mathbb{E}[X] = 1/\lambda$。

- 第一步：$\mathbb{E}[X] = 1/\lambda$，即 $g(\lambda) = 1/\lambda$
- 第二步：令 $\bar{X} = 1/\hat{\lambda}$，解得 $\hat{\lambda} = 1/\bar{X}$

### 6.3 实例：二项分布的概率参数

$X_1, \ldots, X_n \overset{\text{iid}}{\sim} B(10, p)$，要估计 $p$。

- 第一步：$\mathbb{E}[X] = 10p$
- 第二步：令 $\frac{1}{n}\sum X_i = 10\hat{p}$，解得 $\hat{p} = \frac{1}{10n}\sum X_i$

**代入实际数据**：若 $(x_1, \ldots, x_5) = (5, 7, 3, 6, 8)$：
$$\hat{p} = \frac{5 + 7 + 3 + 6 + 8}{10 \times 5} = \frac{29}{50} = 0.58$$

### 6.4 MoM 的优缺点

| 优点 | 缺点 |
|---|---|
| 在温和条件下一致（由大数定律保证） | 一般**不**无偏 |
| 不需要知道完整的数据生成过程，只需知道矩 | 不一定有最小 MSE |
| 对分布假设偏差具有鲁棒性 | |

## 七、期中真题练习

### 7.1 真题 1：样本量与估计精度

**题目**：模拟估计比例参数 $p$。当 $n = 200$ 时，80% 的估计值误差 ≤ 6%。要使 80% 的估计值误差 ≤ 3%，需要多大的 $n$？

**答案**：$n = 800$

**推理**：
- 精度翻倍（误差减半：6% → 3%）
- 由 CLT，$\hat{p}$ 的标准差 $\propto 1/\sqrt{n}$
- 精度翻倍 → 需要 $n$ 变为 4 倍：$200 \times 4 = 800$

### 7.2 真题 2：未知样本量的 Widgets 问题

**背景**：10 次等量抽样，$X_i \sim B(N, p)$，$N$ 未知，$p$ 未知。观测值为 180, 182, 181, 178, 186, 183, 183, 183, 179, 181。

**(a) 为什么不能用 $\bar{X}$ 估计 $p$？**

$p$ 是概率，取值在 $[0, 1]$；$\bar{X}$ 在 180 附近，量纲和范围完全不对。

**(b) 用 MoM 证明 $\hat{P} = 1 - S^2/\bar{X}$ 是 $p$ 的估计量**

- $\text{Var}(X) = Np(1-p)$，由大数定律 $S^2 \approx Np(1-p)$
- $\mathbb{E}[X] = Np$，由大数定律 $\bar{X} \approx Np$
- 因此 $S^2/\bar{X} \approx \frac{Np(1-p)}{Np} = 1 - p$
- 所以 $\hat{P} = 1 - S^2/\bar{X} \approx p$ ✓

**(c) 当 $p$ 很小时的潜在问题？**

$p$ 很小时，可能出现 $S^2/\bar{X} > 1$，导致 $\hat{P} < 0$，估计值为负数，不合理。

**(d) 用 MoM 证明 $\hat{N} = \bar{X}/\hat{P}$ 是 $N$ 的估计量**

- 由 (b) 已知 $\hat{P} \approx p$
- $\bar{X} \approx Np$
- 因此 $\hat{N} = \bar{X}/\hat{P} \approx Np/p = N$ ✓

## 八、置信区间引入（Confidence Intervals）

### 8.1 动机

**钓鱼例子**：估计 Lake Mendota 的鱼数量为 10,848 条。但鱼的数量每天在变化，给出单一点估计不够充分。我们希望给出一个**区间**，以某种概率包含真实值。

### 8.2 核心思想

若估计量 $S(X_1, \ldots, X_n)$ 是 $\theta$ 的无偏估计，$S$ 是随机变量，有一个分布。如果我们知道 $S$ 的分布，就可以找到两个分位数 $L, U$ 使得：

$$\Pr(L \leq S \leq U) = 1 - \alpha$$

### 8.3 示例

$S = \bar{X}$，$n = 25$，$X_i \sim N(0, 1)$。

- $\mathbb{E}[S] = \mu = 0$（此处原设定 $\mu = 0$ 但 slides 用 $\mu = 1$，以 slides 为准）
- $S \sim N(1, 1/25)$
- 用 R 计算 2.5% 和 97.5% 分位数：

```r
qnorm(c(0.025, 0.975), mean=1, sd=1/5)
## [1] 0.6080072 1.3919928
```

- 得到 95% 置信区间 $C = (0.608, 1.392)$
- 即 $\Pr(0.608 \leq S \leq 1.392) = 0.95$

**增大样本量的效果**：$n = 100$ 时，$S \sim N(1, 1/100)$：
```r
qnorm(c(0.025, 0.975), mean=1, sd=1/sqrt(100))
## [1] 0.8040036 1.1959964
```
区间更窄 → 更多数据 → 更高精度 → 更窄的置信区间。

### 8.4 区间估计的严格定义

给定 $X_1, \ldots, X_n \overset{\text{iid}}{\sim} P$，构造两个统计量 $L(X_{1:n})$ 和 $U(X_{1:n})$，使得：

$$\mathbb{P}_{X_1, \ldots, X_n \sim P}(L(X_{1:n}) \leq \theta \leq U(X_{1:n})) = 1 - \alpha$$

区间 $C(X_1, \ldots, X_n) = (L(X_1, \ldots, X_n),\ U(X_1, \ldots, X_n))$ 即为 $(1-\alpha)$ 置信区间。

### 8.5 ⚠️ 对"概率"的正确理解

> **常见误解**："95% 置信区间 [100, 110] 意味着真实参数 $\mu$ 有 95% 的概率落在 [100, 110] 内"——**这是错误的！**

**正确理解**：
- $\theta$ 是**固定未知常数**，不是随机变量
- **随机性来源于数据**：如果反复抽样，每次都能算出一个置信区间
- "95% 置信" 意味着：在所有这些区间中，**大约 95% 会包含真实的 $\theta$**
- 这是关于统计量 $L$ 和 $U$ 的**函数性质**，而不是关于某个具体区间 [100, 110] 的性质

> **老师强调**：置信水平是 $L, U$ 这两个统计量的性质。一旦你代入了具体数据得到了具体的数字区间，这个区间要么包含 $\theta$ 要么不包含，没有"95% 的概率"可言。

### 8.6 基于模拟的置信区间构造思路（预告）

实际中我们不知道 $S$ 的真实分布。一个思路：

1. 先用数据估计参数 $\hat{\theta} = S(X_1, \ldots, X_n)$
2. 假设 $\hat{\theta}$ 接近真实值，从估计的模型 $\text{DGP}(\hat{\theta})$ 中生成 $B$ 组模拟数据
3. 对每组模拟数据计算统计量 $S$
4. 用模拟得到的 $S$ 的分布来近似真实分布，从而构造置信区间

$$\text{DGP}(\hat{\theta}) \to X^{(1)}_{1:n},\ X^{(2)}_{1:n},\ \ldots,\ X^{(B)}_{1:n} \to S(X^{(1)}_{1:n}),\ S(X^{(2)}_{1:n}),\ \ldots,\ S(X^{(B)}_{1:n})$$
