## 一、假设检验回顾与补充（Hypothesis Testing Recap）

### 1.1 假设检验基本流程

假设检验是一种用数据回答"是/否"问题的统计方法。基本步骤：

1. **建立零假设 $H_0$**（通常对应"否"的答案）和备择假设 $H_1$
2. **构造检验统计量** $T$：用数据计算出的单一汇总量，用来衡量数据有多"极端"
3. **确定 $T$ 在 $H_0$ 下的分布**
4. **计算 p 值**：在 $H_0$ 下，观察到与实际数据同样极端或更极端结果的概率
5. **决策**：若 $p \leq \alpha$（预设显著性水平），则拒绝 $H_0$；否则不拒绝

> 老师口头补充：拒绝域（rejection region）的选取和 p 值判断是等价的——统计量落入拒绝域 ⟺ p 值 ≤ α。

### 1.2 Type I / Type II Error 与 Power

| | $H_0$ 为真 | $H_0$ 为假 |
|---|---|---|
| **不拒绝** | 正确（True Negative） | Type II Error（漏检） |
| **拒绝** | Type I Error（误判） | 正确（True Positive） |

- **Type I Error**：$H_0$ 为真却拒绝了，概率控制在 $\alpha$ 以下
- **Type II Error**：$H_0$ 为假却没拒绝
- **Power（功效）**= 1 − P(Type II Error)：$H_1$ 为真时拒绝的概率，越大越好
- 在保证 Type I Error ≤ α 的前提下，我们希望最大化 power

### 1.3 离散分布下的离散化问题

以抛硬币为例：$X_1, \ldots, X_{200} \sim \text{Bernoulli}(p)$，检验 $H_0: p = 0.5$。

- 检验统计量 $T$ = 正面次数 ~ $\text{Binomial}(200, p)$
- 因为 $T$ 是离散分布，CDF 是阶梯函数，可能找不到恰好使 $F_T(t) = 1 - \alpha$ 的临界值
- 例如 `qbinom(0.95, 200, 0.5)` = 112，但 $F_T(112) \approx 0.9615$，不精确等于 0.95
- 实际操作中选取最接近的临界值（如 111），使得拒绝概率近似为 $\alpha$

> 老师强调：在大多数情况下，只要离散分布足够"细"，我们总能找到一个使拒绝概率近似为 α 的临界值。

### 1.4 单侧检验 vs 双侧检验

**单侧检验**（$H_1: p > 0.5$）：
- 拒绝域在右侧：$T > 111$ 时拒绝
- Monte Carlo 模拟验证：$p = 0.5$ 时拒绝率 ≈ 5.34%（Type I Error 受控）
- $p = 0.75$ 时拒绝率 = 1（power 极高）
- $p = 0.25$ 时拒绝率 = 0（设计如此——单侧检验只针对一个方向优化 power）

**双侧检验**（$H_1: p \neq 0.5$）：
- 拒绝域在两侧：$T < 87$ 或 $T > 114$ 时拒绝
- 将 $\alpha$ 均分为两半，各 $\alpha/2$
- $p = 0.5$ 时拒绝率 ≈ 5%
- 无论 $p$ 偏大还是偏小，拒绝率都会快速上升

**比较**：
- 单侧检验在其对应方向上 power 更高
- 双侧检验在两个方向都有检测能力，但单方向的 power 略低于对应的单侧检验

### 1.5 检验统计量的选择并非唯一

老师举了一个生动例子：200 次抛硬币，假设前 100 次全是正面、后 100 次全是反面。

- 用 $T$ = 正面总数 → $T = 100$，看起来完全不极端
- 但直觉上这种数据明显异常

这说明：将高维数据压缩成一维统计量时，必然丢失信息。不同的检验统计量捕捉不同类型的"极端"。例如可以用"最长连续正面次数"作为另一个检验统计量。

### 1.6 α 的选取取决于应用场景

老师以**癌症筛查**为例：

- $H_0$：患者没有癌症
- Type I Error（误诊有癌症）→ 患者焦虑、额外检查，但后果相对可控
- Type II Error（漏诊癌症）→ 后果严重，可能危及生命

因此在这种场景下应适当提高 $\alpha$（容忍更多 Type I Error），以降低 Type II Error。**α 的选择本质上是对两类错误的权衡**。

## 二、点估计（Point Estimation）

### 2.1 什么是估计？

**估计（Estimation）** 是另一个核心统计任务：用数据对某个未知量给出一个猜测值（或值的范围）。

- 与假设检验不同：假设检验回答"是/否"，估计给出一个"数值猜测"
- 背景相同：存在未知的数据生成过程（总体 $P$），我们通过采样获取数据，利用数据推断总体参数

### 2.2 核心概念与符号

| 概念 | 含义 | 符号 |
|---|---|---|
| **总体 / 数据生成过程** | 未知的概率分布 | $P$ |
| **目标参数** | 我们想估计的总体特征 | $\theta(P)$ 或 $\theta$ |
| **样本 / 数据** | 从总体中抽取的 i.i.d. 随机变量 | $X_1, \ldots, X_n \sim_{\text{iid}} P$ |
| **估计量（Estimator）** | 样本的函数（随机变量） | $\hat{\theta}(X_1, \ldots, X_n)$ |
| **估计值（Estimate）** | 代入实际观测值后的具体数值 | $\hat{\theta}(x_1, \ldots, x_n)$ |
| **抽样分布** | 估计量作为随机变量的分布 | — |

> 老师反复强调：**估计量必须是数据的函数，不能包含未知参数**。例如想估计 $\mu$，$\mu$ 本身不是合法的估计量，因为我们算不出 $\mu$。

**估计量 vs 估计值**：
- 估计量 $\hat{\mu} = \bar{X} = \frac{1}{n}\sum X_i$ 是抽象函数（随机变量）
- 估计值 $\hat{\mu} = 172.1$ cm 是代入实际数据后的具体数字

### 2.3 直观例子：估计人类平均身高

- 总体：全球所有人的身高分布 $P$
- 目标参数：$\mu = \mathbb{E}[X]$（平均身高）
- 数据：随机抽取几千人，测量身高 $x_1, \ldots, x_n$
- 估计量：$\hat{\mu} = \bar{X} = \frac{1}{n}\sum_{i=1}^n X_i$
- 估计值：$\hat{\mu} = 172.1$ cm

为什么这是好的估计量？
1. **可行性**：$\hat{\mu}$ 是数据的函数，可以计算
2. **大数定律**：当 $n$ 足够大时，$\bar{X}$ 会接近真实的 $\mu$

### 2.4 哪些是合法的估计量？（课堂练习）

给定 $X_1, \ldots, X_n \sim_{\text{iid}} P$，想估计 $\mu$。以下哪些是合法估计量？

- ✅ $X_1$（虽然只用了一个样本，但它是数据的函数，且 $\mathbb{E}[X_1] = \mu$，无偏）
- ✅ $\frac{X_1 + X_2}{2}$（两个样本的平均，也是合法且无偏的）
- ✅ $\bar{X} = \frac{1}{n}\sum X_i$（最常用，合法且无偏）
- ❌ $\mu$（不是数据的函数，无法从样本计算得到）

## 三、Universal Widgets of Madison 例子

### 3.1 问题设定

UW-Madison 公司生产 widgets，每个 widget 独立地以概率 $p$ 功能正常、概率 $1-p$ 有缺陷。

- **模型**：$X_i \sim \text{Bernoulli}(p)$，i.i.d.
- **目标**：估计 $p$

### 3.2 构造估计量

由于 $\mathbb{E}[X_i] = p$，自然用样本均值估计：

$$\hat{p} = \frac{1}{n}\sum_{i=1}^n X_i$$

由大数定律，$n \to \infty$ 时 $\hat{p} \to p$。

### 3.3 模拟验证（R 代码思路）

```r
# 生成一次实验
generate_widgets <- function(n, p) {
  return(rbinom(1, size=n, p))
}

# Monte Carlo：重复 1000 次实验
n <- 200; p <- 0.8; NMC <- 1000
functional_widgets <- rep(NA, NMC)
for(i in 1:NMC) {
  functional_widgets[i] <- generate_widgets(n, p)
}
hist(functional_widgets / n)   # 抽样分布直方图
abline(v=p, col='red', lwd=4)  # 真实值
```

结果：直方图集中在 $p = 0.8$ 附近，说明估计量效果好。

### 3.4 样本量增大 → 估计更准确

模拟不同 $n$（100, 200, 400, 800）的抽样分布：随着 $n$ 增大，分布越来越集中在真实值 $p$ 周围。

## 四、好的估计量的标准

### 4.1 无偏性（Unbiasedness）

$\hat{\theta}$ 是 $\theta$ 的**无偏估计量**，如果对所有感兴趣的 $P$：

$$\mathbb{E}_{X_1,\ldots,X_n \sim P}\left[\hat{\theta}(X_1, \ldots, X_n)\right] = \theta(P)$$

**偏差（Bias）** 定义为 $\mathbb{E}\hat{\theta} - \theta$。

例子：$\hat{p} = \frac{1}{n}\sum X_i$ 是 $p$ 的无偏估计量，因为：

$$\mathbb{E}\hat{p} = \mathbb{E}\frac{1}{n}\sum X_i = \frac{1}{n}\sum \mathbb{E}X_i = \frac{1}{n}\sum p = p$$

> $\hat{\mu} = \bar{X}$ 对任何均值为 $\mu$ 的分布都是无偏的，利用了期望的线性性。

### 4.2 方差（Variance）

$$\text{Var}(\hat{p}) = \mathbb{E}(\hat{p} - p)^2$$

衡量估计量围绕真值的"散布程度"。

**推导**（以 Bernoulli 为例）：

$$\text{Var}(\hat{p}) = \text{Var}\left(\sum_{i=1}^n \frac{X_i}{n}\right) = \sum_{i=1}^n \text{Var}\left(\frac{X_i}{n}\right)$$

（独立随机变量之和的方差 = 方差之和）

$$\text{Var}\left(\frac{X_1}{n}\right) = \mathbb{E}\left(\frac{X_1 - p}{n}\right)^2 = \frac{(1-p)^2 \cdot p + p^2 \cdot (1-p)}{n^2} = \frac{p(1-p)}{n^2}$$

因此：

$$\boxed{\text{Var}(\hat{p}) = n \cdot \frac{p(1-p)}{n^2} = \frac{p(1-p)}{n}}$$

**关键结论**：
- 方差以 $1/n$ 的速率递减
- 要将方差减半 → 样本量翻倍

### 4.3 标准差（Standard Deviation）

方差的单位是"平方单位"，不直观。标准差更合适：

$$\text{sd}(\hat{p}) = \sqrt{\text{Var}(\hat{p})} = \frac{\sqrt{p(1-p)}}{\sqrt{n}}$$

- 标准差以 $1/\sqrt{n}$ 递减
- **要将标准差减半 → 样本量需要增大 4 倍**

### 4.4 "坏"事件的概率

能否 100% 保证 $\hat{p}$ 任意接近 $p$？**不能**。

例如：所有 $n$ 个 widget 都正常（$\hat{p} = 1$）的概率为 $p^n > 0$。虽然极小，但始终为正。

不过，随着 $n$ 增大，估计值远离真值的概率会越来越小。

### 4.5 相合性（Consistency）

当 $n \to \infty$，$\hat{\theta}(X_1, \ldots, X_n)$ "收敛"到 $\theta$。

**弱大数定律**：若 $X_1, X_2, \ldots$ i.i.d.，均值为 $\mu$，则对任意 $\epsilon > 0$：

$$\lim_{n \to \infty} \Pr\left[\left|\frac{1}{n}\sum_{i=1}^n X_i - \mu\right| > \epsilon\right] = 0$$

等价地：对任意 $\epsilon > 0, \delta > 0$，存在 $n_0$ 使得 $n \geq n_0$ 时：

$$\Pr\left[\left|\frac{1}{n}\sum_{i=1}^n X_i - \mu\right| > \epsilon\right] \leq \delta$$

### 4.6 均方误差（MSE）

$$\text{MSE}(\hat{\theta}) = \mathbb{E}\left[\left(\hat{\theta} - \theta\right)^2\right]$$

### 4.7 好的估计量的三个标准总结

1. **无偏（Unbiased）**：$\mathbb{E}\hat{\theta} = \theta$
2. **相合（Consistent）**：$n \to \infty$ 时 $\hat{\theta} \to \theta$
3. **MSE 小**

### 4.8 注意：两种理解"参数"的方式

老师提到两种直觉：
1. 获取无穷多数据时 $S(X_1, \ldots, X_n)$ 的极限
2. 重复实验多次，$S_i$ 的平均值 $\mathbb{E}[S(X_1, \ldots, X_n)]$

**这两者一般不同！** 例如 $S = \max\{X_1, \ldots, X_n\}$ 或 $S = \min\{X_1, \ldots, X_n\}$，极限和期望是不同的。

## 五、矩估计法（Method of Moments）简介

### 5.1 指数分布的 rate 参数估计

$X_1, \ldots, X_n \sim \text{Exp}(\lambda)$，$\mathbb{E}[X_1] = 1/\lambda$。

**思路**：
1. 参数用矩表达：$\lambda = 1 / \mathbb{E}[X]$
2. 大数定律：$\bar{X} \approx \mathbb{E}[X] = 1/\lambda$
3. 估计量：$\hat{\lambda} = 1 / \bar{X}$

```r
run_exprate_expt <- function(n, rate) {
  data <- rexp(n=n, rate=5)
  Xbar <- mean(data)
  return(1/Xbar)
}
```

Monte Carlo 模拟结果：直方图集中在真实值 $\lambda = 5$ 附近。

### 5.2 矩估计的一般思想

1. 将目标参数 $\theta$ 表达为分布矩的函数：$\theta = g(\mathbb{E}X, \mathbb{E}X^2, \ldots)$
2. 用样本矩替代总体矩：$n^{-1}\sum X_i \approx \mathbb{E}X$，$n^{-1}\sum X_i^2 \approx \mathbb{E}X^2$，...
3. 估计量：$\hat{\theta} = g(\bar{X}, \overline{X^2}, \ldots)$

> 老师提到：后续还会学习**最大似然估计（MLE）**等其他构造估计量的方法。

## 六、课堂练习

$X_1, \ldots, X_n \sim N(\mu, \sigma^2)$，想估计 $\mu^2$。

考虑 $T(X_1, \ldots, X_n) = X_1^2$，问：$T$ 是否无偏？偏差是多少？

$$\text{Bias} = \mathbb{E}[X_1^2] - \mu^2 = \left(\text{Var}(X_1) + (\mathbb{E}X_1)^2\right) - \mu^2 = \sigma^2 + \mu^2 - \mu^2 = \sigma^2$$

所以 $T = X_1^2$ **不是** $\mu^2$ 的无偏估计量，偏差为 $\sigma^2$。

## 七、预告

- 有限样本下估计的精度 → **置信区间（Confidence Intervals）**
- 大数定律只说"$n$ 足够大时接近"，但具体多大算够？→ 需要不确定性量化
- 更多估计量构造方法：最大似然估计（MLE）
