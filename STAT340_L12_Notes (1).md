## 一、区间估计概述

### 1.1 什么是区间估计

区间估计与点估计类似，但我们不再给出一个单一的猜测值，而是构造一个**区间猜测**来估计未知参数 $\theta$。

给定数据 $X_1, \ldots, X_n \overset{iid}{\sim} P$，目标是找到两个统计量（均为数据的函数）：$L(X_{1:n})$ 和 $U(X_{1:n})$，使得：

$$\mathbb{P}(L(X_{1:n}) \le \theta \le U(X_{1:n})) = 1 - \alpha$$

**Yongyi 强调**：这个区间是"here and there"的——由于数据的随机性，每次得到的区间不同，但我们希望其中 $1 - \alpha$（例如 95%）的区间能覆盖真值。

### 1.2 核心思路

我们已经有了点估计量 $S(X_1, \ldots, X_n)$（假设是无偏的，即 $\mathbb{E}[S] = \theta$）。为了构造置信区间，我们不仅需要 $S$ 的值准确，还需要了解 $S$ 的**抽样分布有多分散**——即量化不确定性。

## 二、模拟法构造置信区间（Simulation-based CI）

### 2.1 理想情况（已知 DGP）

如果我们知道真实的数据生成过程（DGP），可以：
1. 从真实 DGP 反复生成大小为 $n$ 的独立数据集
2. 对每个数据集计算统计量 $S$
3. 用这些 $S$ 的直方图 / 分位数来构造 CI

### 2.2 实际情况（DGP 未知）

**实际中我们并不知道真实 DGP。** 解决方案是**用数据估计 DGP**：

**Step 1**：从观测数据中估计 DGP 的参数（形成 $\hat{\theta}$），得到"假的 DGP"（fake DGP / approximated DGP）

**Step 2**：假装 $\hat{\theta}$ 就是真值，从估计的 DGP 中反复生成大量"假数据集"（fake datasets），每个数据集大小为 $n$

**Step 3**：对每个假数据集计算统计量 $S$，得到 $B$ 个（如 2000 个）$S$ 值

**Step 4**：取这些 $S$ 值的 **2.5% 和 97.5% 分位数**，构成 95% 置信区间

> **Yongyi 澄清**：Step 1 中用于估计 DGP 的 $\hat{\theta}$ 和 Step 3 中计算的统计量 $S$ **可以相同也可以不同**——它们服务于不同目的。$\hat{\theta}$ 是为了估计 DGP 以生成假数据，$S$ 是我们想要研究其抽样分布的统计量。

### 2.3 两层近似（Two Layers of Approximation）

**Yongyi 特别强调**：模拟法只产生**近似正确的**置信区间，有两层近似误差：

**第一层**：我们用的是**假 DGP**（estimated DGP），不是真实 DGP。真实 DGP 的参数未知，我们只能用 $\hat{\theta}$ 近似。如果 $n$ 太小（如只有 10 或 20），$\hat{\theta}$ 可能离真值较远，假 DGP 就不够准确。

**第二层**：即使 DGP 是真的，我们也只能模拟**有限次**（$B$ 次），得到有限个 $S$ 值。这个有限直方图只是真实抽样分布的近似。$B$ 越大越好，但永远不是无穷大。

> **实际建议**（代课老师 Bi 补充）：$B$（MC 重复次数）应该设到你的电脑能在几秒内跑完的最大值。注意区分 $n$（每个实验的样本量，由问题决定，固定不变）和 $B$/$M$（MC 重复次数，越大越好）。

### 2.4 Poisson 率参数示例

设 $X_1, \ldots, X_n \overset{iid}{\sim} \text{Poisson}(\lambda)$，目标：构造 $\lambda$ 的 95% CI。

**Step 1**：$\hat{\lambda} = \bar{X}$（因为 $\mathbb{E}[X_i] = \lambda$）

**Step 2-3**：从 $\text{Poisson}(\hat{\lambda})$ 生成 $B = 2000$ 组假数据（每组 $n = 80$），每组算 $\hat{\lambda}^{(b)} = \bar{X}^{(b)}$

**Step 4**：取分位数

```r
CI <- quantile(replicates, probs = c(0.025, 0.975))
## 2.35  3.0625
```

真实 $\lambda = 2.718$ 落在此区间内。

**验证**：重复整个过程 $N$ 次（包括从真实 DGP 重新抽样、估计 DGP、生成假数据、构造 CI），检查覆盖率 ≈ **0.946**，接近 95%。

> **Yongyi 强调关于重复整个过程**：这里是把从抽样到构造 CI 的**所有步骤**（包括生成 $B$ 次假数据）完全独立地重复 $N$ 次，看 CI 覆盖真值的比例。

### 2.5 指数分布率参数示例

$X_1, \ldots, X_n \overset{iid}{\sim} \text{Exp}(\lambda)$，$\mathbb{E}[X] = 1/\lambda$，所以 $\hat{\lambda} = 1/\bar{X}$。

模拟 CI 步骤：
1. $\hat{\lambda} = 1/\bar{X}$ → 假 DGP 为 $\text{Exp}(\hat{\lambda})$
2. 从 $\text{Exp}(\hat{\lambda})$ 反复生成假数据，每组算 $1/\bar{X}^{(b)}$
3. 取分位数

### 2.6 影响近似质量的因素

- **$n$ 太小**（如 10 或 20）→ $\hat{\theta}$ 不准确，假 DGP 偏离真实 DGP，CI 覆盖率下降
- **$\lambda$ 极端**（接近 0 或极大）→ 分布变得"不友好"，需要更大 $n$ 才能准确估计
- **$B$ 太小** → 抽样分布的近似不够精确

## 三、基于 CLT 的置信区间

### 3.1 中心极限定理（CLT）

若 $X_1, X_2, \ldots$ 为 iid 随机变量，$\mu = \mathbb{E}[X_1]$，$\sigma^2 = \text{Var}(X_1)$，则当 $n \to \infty$：

$$\frac{\bar{X} - \mu}{\sqrt{\sigma^2 / n}} \xrightarrow{d} N(0, 1)$$

**Yongyi 解释"收敛于分布"的含义**：对于每个 $n$，标准化的 $\bar{X}$ 是一个随机变量，有自己的概率分布。当 $n = 1$ 时分布可能很歪，$n = 10$ 时更平滑一些，$n = 100$ 时接近钟形。CLT 说的是这一系列分布**收敛到**标准正态分布——不管原始 $X_i$ 是什么分布（指数、二项、甚至混合分布都行）。

**R 代码验证**（以 $\text{Exp}(1)$ 为例）：

| $n$ | 标准化 $\bar{X}$ 的形状 |
|-----|-------------------------|
| 10  | 明显右偏 |
| 15  | 偏斜减小 |
| 20  | 接近对称 |
| 30  | 很像正态 |

### 3.2 用 CLT 构造 CI

由 CLT，对于 $Z \sim N(0,1)$：

$$0.95 = \Pr[-1.96 \le Z \le 1.96] \approx \Pr\!\left[-1.96 \le \frac{\bar{X} - \mu}{\sqrt{\sigma^2/n}} \le 1.96\right]$$

代数变换后：

$$\boxed{\bar{X} \pm 1.96 \sqrt{\frac{\sigma^2}{n}}}$$

是 $\mu$ 的**近似 95% 置信区间**（前提：已知 $\sigma^2$）。

**若 $\sigma^2$ 未知**（通常如此）：用样本方差 $S^2 = \text{var}(\text{data})$ 代替，仍然得到有效的近似 CI。

> **Yongyi**："在作业中你们会练习估计 $\sigma^2$ 的情况。只要 $\hat{\sigma}^2$ 是一致估计量，plug in 之后 CI 仍然有效。"

### 3.3 几何分布均值的 CLT CI 示例

$X_1, \ldots, X_n \overset{iid}{\sim} \text{Geometric}(p)$，均值 $= (1-p)/p$，方差 $= (1-p)/(np^2)$。

CLT 给出标准化统计量：

$$\frac{\bar{X} - (1-p)/p}{\sqrt{(1-p)/(n \cdot p)}} \;\dot{\sim}\; N(0,1)$$

**实际计算**（数据 $n = 30$, $p = 0.33$）：

```r
data <- c(2, 0, 1, 1, 0, 3, 2, 2, 0, 0, 1, 6, 0, 1, 0, 0, 0,
          11, 0, 1, 0, 1, 0, 2, 0, 2, 1, 2, 1, 0)
xbar <- mean(data)       # 1.333
varhat <- var(data)      # 4.989
n <- length(data)
c(xbar - 1.96*sqrt(varhat/n), xbar + 1.96*sqrt(varhat/n))
## 0.534  2.133
```

真实均值 $(1-0.33)/0.33 = 2.030$ 落在 CI 内。

## 四、两种 CI 方法的比较

| 特性 | 模拟法 | CLT 法 |
|------|--------|--------|
| **通用性** | 高——不限于特定分布族或估计量形式 | 需要样本均值或类似结构 |
| **直觉性** | 强——直接通过模拟理解不确定性 | 需要理解正态近似 |
| **计算** | 需要 MC 模拟，计算量大 | 封闭公式，快速 |
| **适用条件** | 假 DGP 近似准确 + $B$ 足够大 | $n$ 足够大使 CLT 近似合理 |

> **Yongyi**："在实践中，你需要根据具体应用选择哪种更好、更容易、更可行。"

## 五、检验与置信区间的对偶性（Duality）

### 5.1 从 CI 到假设检验

**实际背景**（Yongyi 举例）：在检测治疗效果时，研究者往往想知道效果是否显著（$\neq 0$），但报告的形式通常是置信区间。这正是因为对偶性——看 CI 是否包含 0 就等同于做了假设检验。

设 $C = (L, U)$ 是 $\theta$ 的 $(1-\alpha)$ 置信区间。

**拒绝规则**：计算 CI $C$。若 $\theta_0 \notin C$，拒绝 $H_0: \theta = \theta_0$；否则不拒绝。

**为什么此规则的 Type I error = $\alpha$**：

$$\Pr(\text{拒绝} \mid H_0) = \Pr(\theta_0 \notin C \mid \theta = \theta_0) = 1 - \Pr(\theta_0 \in C \mid \theta = \theta_0) = 1 - (1-\alpha) = \alpha$$

> **Yongyi**："这个论证虽然看起来涉及很多符号，但逻辑其实非常简单——直接由置信区间的定义得出。"

### 5.2 从假设检验到 CI

反方向也成立：对于任意 $\theta_0$，如果能进行水平 $\alpha$ 的检验 $H_0: \theta = \theta_0$，则收集所有**不被拒绝**的 $\theta_0$ 值，即构成 $(1-\alpha)$ 置信区间。

## 六、课堂练习与考试题

### 练习 1（四选一）

以下哪项为真？

**A.** 对于来自固定分布的 iid 样本，95% CI 的宽度随样本量增大保持不变。
→ **错误**。CI 宽度与 $\sqrt{\sigma^2/n}$ 成比例，$n$ 增大时宽度缩小。

**B.** "总体均值的 99% CI 为 (50, 60)" 表示样本均值有 99% 概率在 50-60 之间。
→ **错误**。Yongyi 详细解释："置信区间性质是关于**数据函数** $L(\text{data}), U(\text{data})$ 的，不是关于某次具体算出的 (50, 60)。换一组数据，CI 就变成别的数了。不能说 $\mu$ 有 99% 概率在 (50, 60) 内——随机性在数据里，不在参数上。"

**C.** 假设检验中，Type II error 发生在零假设被错误拒绝时。
→ **错误**。这描述的是 Type I error。Type II error 是零假设为假但未被拒绝。

**✅ D.** 显著性水平 $\alpha$ 的假设检验即使备择假设 $H_1$ 改变也仍然有效。
→ **正确**。$\alpha$ 只依赖于 $H_0$ 下的分布，与 $H_1$ 无关。

### 练习 2（FA25 期中）

$n = 16$，几何分布，$p$ 的 95% CI = (0.456, 0.563)。

- "$p$ 有 95% 概率在此区间" → **错**（$p$ 是固定常数）
- "样本翻倍，区间约变窄一半" → **近似正确但不精确**（宽度按 $1/\sqrt{n}$ 缩放，翻倍 $n$ 只缩到 $1/\sqrt{2} \approx 0.707$ 倍）
- ✅ "用同样数据对 $H_0: p = 0.5$ 做双侧检验不拒绝" → **正确**（$0.5 \in (0.456, 0.563)$，由对偶性）
- "再取样有 95% 概率与此区间重叠" → **错**
- "$\bar{x} \in (0.456, 0.563)$" → **不一定**

### 练习 3（S25 期中）

$n = 20$，Poisson，10,000 次模拟 MC CI = (0.45, 0.72)。

- **✅ a.** 随机选一个 MC 模拟的点估计，有 95% 概率在 (0.45, 0.72) 内 → **正确**（MC CI 就是取中间 95% 的模拟估计值）
- b. "$\lambda$ 有 95% 概率在 0.45-0.72" → **错**（$\lambda$ 是固定的）
- c. "$\lambda > 0.72$ 的概率为 2.5%" → **错**（同上）
- d. "重新抽样，$\bar{X}$ 有 95% 概率在此区间" → **错**
- e. "100,000 次 MC 会让 CI 更窄" → **错**（MC 次数主要影响分位数估计精度，不显著改变 CI 宽度；$n$ 才是关键）

## 七、关键总结

1. **区间估计的本质**：一个随机的、数据依赖的区间。其性质（覆盖率）是关于**重复抽样过程**的，不是关于某次具体算出的数值。

2. **模拟法 CI**：
   - 步骤：估计 DGP → 生成假数据 → 计算统计量 → 取分位数
   - 两层近似：① 假 DGP ≠ 真 DGP ② 有限次模拟 ≠ 真实抽样分布
   - 通用但需要 $n$ 足够大和 $B$ 足够大

3. **CLT CI**：$\bar{X} \pm z_{\alpha/2}\sqrt{\hat{\sigma}^2/n}$
   - 简单快速，有封闭形式
   - 需要 $n$ 足够大使正态近似合理
   - $\sigma^2$ 未知时用样本方差代替

4. **对偶性**：$(1-\alpha)$ CI 与水平 $\alpha$ 检验可互相转换
   - CI 包含 $\theta_0$ ⟺ 不拒绝 $H_0: \theta = \theta_0$
   - 实际应用：看治疗效果的 CI 是否包含 0 就知道效果是否显著

5. **常见误解**：
   - ❌ "$\theta$ 有 95% 概率在 CI 内"
   - ✅ "如果重复整个过程，95% 的 CI 会覆盖 $\theta$"

6. **精度与样本量**：误差按 $1/\sqrt{n}$ 缩放，减半误差需 4 倍样本
